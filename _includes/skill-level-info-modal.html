What do the skill levels mean?

Indicates I have either dabbled with the technology (nothing more than experimentation for a few days/weeks) or have used it a long time ago.

Novice

Indicates I have used the technology in a project before, but I will require access to documentation, examples, and some time to master it.

Intermediate

Indicates I have used the technology in a project before and can confidently work with it without constant access to documentation, examples, and help.

Advanced

Indicates I have used the technology in a project before and can confidently work with it without access to examples and help. However, I still need access to documentation (or source code) to effectively use it, as I do not memorize this information.
